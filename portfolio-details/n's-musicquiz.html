Project information

  • Github / Youtube
  • 진행기간: 2022/01/22 ~ 2022/02/26
  • 사용기술: Java, Python, Tomcat, Swing
  • 요약

담당한 기능

  • 파이썬을 이용하여 유튜브링크에서 mp3파일 추출 후 1초 5초 10초로 나누어서 저장함
  • 초기계획은 유튜브링크와 연동하려고 하였으나 추후 링크가 없어질경우 버그가 생길 수 있어 따로 서버에 음악 파일을 저장하고 정보 데이터를 JSON파일로 만들어서 저장.
  • 버튼이미지와 오버레이 이미지 따로 구분하여 마우스 클릭시 어떤 버튼을 클릭했는지 한 눈에 알 수 있음
  • 싱글파트 정답입력 후 제출시 서버에서 정답과 일치하는지 확인 구현
  • 플레이중 소리 볼륨조절 기능 구현
  • 프로젝트 이후 코드 리팩토링 : 흩어진 변수를 관리하기 위한 R 클래스 생성

성장한 부분

  • tomcat 서버와의 데이터 전송을 구현할 수 있다
  • 자바 멀티쓰레드를 구현할 수 있다.
  • 자바에서 JSON파일 데이터를 다룰 수 있다.

아쉬운 부분

  • 서버를 단순히 껏다키기만 할 뿐 따로 프레임을 구현하지 못하였다.
  • 노래가 추가되었을 때 클라이언트가 가지고 있는 노래리스트와 비교하여 추가하는 기능을 구현하지 못하였다.
