Project information

  • 진행기간: 2021/04/01 ~ 2021/04/20

  • 요약
  • 보호자가 병원 방문을 하지 않더라도 입원한 환자의 건강 상태 확인

담당한 기능

  • 로지스틱 회귀분석을 이용하여 환자 질병 발생 예측
  • 32개의 환자정보속성과 26개의 질병 발생 여부가 있는 과거 환자 데이터 상관계수를 추출하여 상위 30% 총 10개의 환자정보속성 값을 사용

성장한 부분

  • 여러 개의 데이터테이블의 기본키와 외래키를 이용하여 원하는 데이터를 추출할 수 있다
  • 데이터를 이용하여 간단한 전처리 후 회귀분석이 가능하다
  • DML, DDL를 이해하고 사용할 수 있다.

아쉬운 부분

  • 환자 데이터를 이용하여 분석했지만 데이터가 질병을 걸렸던 환자의 데이터라 학습이 제대로 수행되지 못하였다.
