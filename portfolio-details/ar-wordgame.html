Project information

  • Github
  • 진행기간: 2022/05/12 ~ 2022/05/24
  • 사용기술: Java, Android, ARCore, MPAndroidChart, SQLi
  • 요약
  • 메타버스(AR)를 결합한 영단어 게임

담당한 기능

  • 배경음악 플레이와 음소거 기능 : 백그라운드됬을때 멈춤, 액티비티간 이동시는 멈추지않고 연속 재생되게 구현
  • 메인화면 카운트 다운 기능 : 게임시작전 준비시간을 주기위해 3초 카운트다운
  • 게임화면 시간계산타이머 : 게임플레이 시간을 체크하기위한 타이머
  • 종료화면 : 기록데이터를 한눈에 보기위한 테이블, 그래프화
  • AR오브젝트 좌표와 터치좌표간의 민감도 조절

성장한 부분

  • 안드로이드 스튜디오에서 AR Core 기술을 사용할 수 있다.
  • 데이터를 테이블로 보여주거나 그래프로 보여줄 수 있다.
  • xml파일은 include를 이용할 수 있다.(카운터다운, 타이머 xml)
  • 안드로이드내에서 DB를 사용할 수 있다.

아쉬운 부분

  • 터치 인식률을 증가시키려고 많은 노력을 하였으나 멀리서는 인식이 어렵고 가까이 다가가서 클릭해야만 인식이 된다.
  • 서버를 열고 멀티게임 기능을 추가못한 것이 아쉽다.
