Project information

  • Github / PDF
  • 진행기간: 2022/06/01 ~ 2022/06/16
  • 사용기술: Java, JSP, Tomcat, MariaDB, GCP
  • 요약
  • 키보드를 판매하는 사이트로 제조사, 종류, 스위치, 가격대등 간단한 필터링 검색이 가능합니다

담당한 기능

  • DB 서버구축
    => GCP에 Cent OS7 환경에 MariaDB서버 구축
  • 검색 필터 리스트 출력
    => 멀티필터 가능
  • KG 이니시스 Rest API를 활용한 주문 결제, 취소
  • 다음 지도 API를 활용한 우편번호 찾기
  • Cafe24 호스팅

성장한 부분

  • 추후 DB가 필요한 팀 프로젝트가 있을 시 DB서버를 구축해서 보다 편한 DB작업이 가능하다.
  • 자바에서 HttpClient로 Rest API를 호출할 수 있다.(결제 취소작업)
  • DB에서 원하는 데이터를 필터링하여 웹으로 출력할 수 있다.(Subquery)
  • XSS 공격 방어를 할 수 있다.

아쉬운 부분

  • 웹 호스팅을 Cent OS 환경에서 tomcat를 설치하여 배포하려고 시도하였지만 실패하고 Cafe24를 이용하여 호스팅하였다.
  • 시간관계상 구현을 못한 기능들이 있다.(휴대폰 인증, 게시글 하루제한 등)
